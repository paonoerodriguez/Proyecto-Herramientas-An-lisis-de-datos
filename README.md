# Proyecto-Herramientas-An-lisis-de-datos
Proyecto final del curso “Herramientas Básicas para el Análisis de Datos” (UTN), enfocado en el análisis de devoluciones de prestaciones de discapacidad de la SSS durante enero y febrero de 2026, con el objetivo de identificar errores frecuentes y su impacto económico.
